Input Feature Class

The Feature Layer datatype lets you pick a layer from your map using the drop-down.

Feature layers represent feature classes | | (Show Fragment) they also allow users to pick a layer from their map using the drop-down

Feature Type Filter

Limit features to lines only.

By using a filter for the Feature Type, we can limit features to lines only

Output Feature Class

Feature class allows user to browse to where they'd like the output to be

Method

This parameter lets the user pick how they would like to shrink the line | String data type since we will be using our own values | Value list filter contains the values that the user will see as options | Percentage is the next parameter, and we are using that as the default

Value List Filter

The user is given two options. You can use the value list filter to add options that you want to limit the user to

Percentage (%)

Range Filter

Limit the parameter to a numerical range.

The range filter allows you to limit the user to a numerical range. In this case, we're limiting it from 0 to 100 since it's a percentage (and we don't want users to try to extend a line)

Length

This is the second method the user can use to shrink a line. It adopts the same unit used by the input feature class.

Source Code


                        import arcpy
                        import os

                        # Initialize parameters
                        in_fc = arcpy.GetParameter(0)
                        out_fc = arcpy.GetParameterAsText(1)
                        method = arcpy.GetParameter(2)
                        percentage = arcpy.GetParameter(3)/100
                        length = arcpy.GetParameter(4)

                        # Create output fc
                        out_ws, out_name = os.path.split(out_fc)
                        arcpy.management.CreateFeatureclass(out_ws, out_name, template=in_fc, 
                                                            spatial_reference=arcpy.Describe(in_fc).SpatialReference)
